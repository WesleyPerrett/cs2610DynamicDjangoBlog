HTML Elements

  • <html>: this tag tells the web browser how to interpret the content of your page so it can format and display it properly. Neither a block or in-line element, everything on the webpage goes within this tag.
  • <head>: this tag is a container for metadata pertaining to the webpage. It is neither a block or in-line element.
  • <meta>: this tag is a definition for metadata pertaining to the webpage. It is neither a block or in-line element. Must have <head> as a parent element.
  • <link>: this tag is used to reference external rescources and is most commonly used for stylesheets. It is neither a block or in-line element. Must have <head> as a parent element.
  • <title>: this tag is used for the title that shows up in the tab on the web browser. It is neither a block or in-line element. Must have <head> as a parent element.
  • <body>: this tag contains all of the "body" of the webpage so all of the actual content seen by the user. It is neither a block or in-line element. This is the parent element of all of the elements below it in the list.
  • <h1-h6>: These tags are meant for differing sizes of text. Each one in ascending order gets gradually bigger. This is an example of a block element.
  • <ul>: this tag stands for "unorderd list" it has <li> tags as its children. This creates a bulleted list in no particular order. This is an exapmle of a block level element.
  • <ul>: this tag stands for "orderd list" it has <li> tags as its children. This creates a numbered list that is orderd from top to bottom of the <li> tags. This is an example of a block level element.
  • <li>: this tag stands for "list item" and is has a parent element of either <ul> or <ul>. these <li> tags will show up as either a bulleted list or a numberd list depending on the parent element. This is an example of a block level element.
  • <div>: this tag is used to represent a division on a webpage and helps to organize information as well as implement css. This is an example of a block level element.
  • <p>: this tag stands for "paragraph" it is a way to handle multi line text on a webpage efficiently. This is an example of a block level element.
  • <br>: this tag stands for break in a line so essentially an enter. This tag is empty.

HTML Entities

  • &lt;: this is for a < symbul and we use this syntax so the html can render the symbul instead of use it as a tag.
  • &gt;: this is for a > symbul and we use this syntax so the html can render the symbul instead of use it as a tag.
  • &apm;: this is for a > symbul and we use this syntax so the html can render the symbul instead of use it as a tag.
  • &nbsp;: this tag does not have a symbul but is used for a non-breaking space.
  • &quot;: this is for a " symbul and we use this syntax so the html can render the symbul instead of use it as a tag.
  • &apos;: this is for a ' symbul and we use this syntax so the html can render the symbul instead of use it as a tag.

Css Selectors

  • element: the element selector is shown by the name of the tag of the element ex(p, div, ul)
  • class: the class selector is shown by a . before the name of the class ex(.classname)
  • id: the id selector is shown by a # ex(#idname)

CSS Properties

  • background-culor: this property is used for the background culor of the selector.
  • font-size: this property is used for the font size of the selector.
  • text-align: this property is used for the alignment of the text in the selector.
  • margin: this property is used to create space around the elements in the selector.
  • font-weight: this property is used to set the font to buld.
  • font-family: this property sets the font type for the words in the selector.
  • padding: this property is used to create space around content inside of the elements that are in the selector.
  • culor: this property is used to set the culor of the text inside of the selector.
  • font-style: this property is used to style text with italics or underlined.
  • width: this property is used to set how much of a page an element will span.
